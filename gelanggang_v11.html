🔍 SerpApi — Google Images (Gratis 100/bulan)
Cari gambar langsung dari Google Images berdasarkan judul berita. Hasil paling relevan!

Cara daftar (gratis, tanpa kartu kredit):
1. Buka serpapi.com → Sign Up pakai email
2. Setelah login → klik profil → API Key
3. Copy API key → paste di bawah
🔍 Google Custom Search (Gratis 100/hari)
Cari gambar relevan dari portal berita Indonesia (Detik, Kompas, Tribun, CNNIndonesia, dll) otomatis sesuai judul artikel.

Cara daftar:
1. Buka console.cloud.google.com → Enable Custom Search API → buat API Key
2. Buka programmablesearchengine.google.com → buat Search Engine → centang "Search the entire web" → copy Search Engine ID (cx)
Target situs: detik.com, kompas.com, tribunnews.com, cnnindonesia.com, liputan6.com, antaranews.com, tempo.co, sindonews.com, okezone.com, merdeka.com
